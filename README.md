# motd puppet module  

## Class: motd  

This class distributes a ``/etc/motd`` to alert users the server is managed by Puppet   

** Only tested on CentOS/RHEL5.  **  

### Parameters:  
	None	

### Actions:  

Distributes a ``/etc/motd``  

### Requires:  
	None
